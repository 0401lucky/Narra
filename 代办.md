# 代办

## 应用内更新

- 排查并修复 `1.0.2 -> 1.0.4` 应用内更新时，下载完成后出现“解析失败 / 找不到安装包”的问题。
- 复盘当前 `DownloadManager.getUriForDownloadedFile()` + `ACTION_VIEW` 安装链路在不同 ROM，尤其 MIUI 上的兼容性。
- 评估并改造为更稳的安装方案：
  使用应用可控路径缓存 APK，再通过 `FileProvider` 生成安装 URI 拉起系统安装器。
- 补充真机回归：
  从旧版本包升级到新版本包，验证“检查更新 -> 下载 -> 安装”整条链路可用。
- 为安装失败补充更明确的错误提示与日志，区分：
  下载任务丢失
  URI 获取失败
  包校验失败
  系统安装器拉起失败
